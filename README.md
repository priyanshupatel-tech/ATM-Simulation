# 💳 ATM System (Python)

A simple ATM simulation system built using Python. This is a command-line based project that allows users to perform basic banking operations.

## 🚀 Features

* Check account balance
* Deposit money
* Withdraw money
* Transaction handling
* Exception Handling

## 🛠️ Technologies Used

* Python

## 📌 How to Run

1. Clone the repository
2. Run the Python file:

   ```
   python atm.py
   ```

## 💡 Learning Outcomes

* Understanding of conditional logic
* Working with functions
* Real-world simulation of ATM operations

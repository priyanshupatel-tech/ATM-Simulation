# 💳 ATM Simulation System

A command-line based **ATM Simulation** built with Python that replicates core banking operations like balance check, deposit, and withdrawal with proper exception handling.

---

## 🚀 Features

- 💰 Check account balance
- ➕ Deposit money
- ➖ Withdraw money with balance validation
- ⚠️ Exception handling for invalid inputs
- 🔄 Transaction loop until user exits

---

## 🛠️ Technologies Used

- **Language:** Python 3
- **Concepts:** Functions, Conditional Logic, Exception Handling, Loops
- **Type:** Command-Line Interface (CLI)

---

## 📌 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/priyanshupatel-tech/ATM_Simulation.git
   cd ATM_Simulation
   ```

2. Run the program:
   ```bash
   python atm.py
   ```
